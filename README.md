- ⚡ **Vite** — Lightning-fast build tool
- 🎨 **Tailwind CSS** — (Optional) Utility-first styling
- 🌐 **Netlify / Vercel** — (Optional) Deployment

- 💼 Projects showcase
- 🛠️ Skills & Tech Stack
