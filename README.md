# Portfolio

A minimalistic and modern portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Responsive design
- Project and experience showcase
- Contact section
- Easy to customize

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/adem-loghmari/portfolio.git
   cd portfolio
   ```
2. Install dependencies:
   ```sh
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```sh
   npm run dev
   # or
   yarn dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```sh
npm run build
# or
yarn build
```
